## Security

I take my software security seriously. If you believe you have found a security vulnerability in any of my own repositories, please report it to me as described below.

## Reporting Security Issues

**Do not report security vulnerabilities through public GitHub issues.**

Instead, please report them to my personal email at [rafael.ghiorzi@gmail.com](mailto:rafael.ghiorzi@gmail.com).

If possible, include a comprehensive message describing the problem so that I can look into it as soon as possible! Try including:

- What repository is the problem in
- What is the problem causing
- What files are problematic
- Potential fixes, if you think of any

## Preferred Languages

You can message me in Portuguese or English, both are fine for me.
